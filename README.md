# Heroes of the Storm — Анализ пиков

Веб-приложение для анализа пиков героев и просмотра базы героев Heroes of the Storm.

## Структура проекта

```
├── html/           — HTML-страницы
├── css/            — стили (оформление в стиле HoTS)
├── img/            — локальные изображения (герои, карты, логотип)
├── js/
│   ├── main.js     — точка входа
│   ├── components/ — Header, Footer, HeroBanner, StatsBar
│   ├── pages/      — AnalyzerPage, HeroesPage, MapsPage, TierlistPage
│   ├── data/       — база героев, ассеты, карты, константы
│   └── utils/      — анализ пиков, профили героев
├── start-server.bat
└── index.html      — редирект на html/index.html
```

## Запуск

Дважды кликните `start-server.bat` или выполните:

```bash
python -m http.server 5500
```

Откройте: http://localhost:5500/html/index.html

## Страницы

- **Анализ пиков** — выбор союзников, противников и карты с рекомендациями по ролям
- **Герои** — список всех героев с описанием, плюсами, минусами, синергиями и контрами
- **Карты** — изображения полей битвы и подходящие герои
- **Тирлист** — герои по мета-тиерам S, A, B, C

Все изображения (портреты, иконки героев и карты) хранятся локально в папке `img/`.
